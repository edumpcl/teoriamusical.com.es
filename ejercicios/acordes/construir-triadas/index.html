CONSTRUIR TRÍADAS EN FUNDAMENTAL

Se muestra la nota del bajo (fundamental) y el tipo de acorde. La tarea es dibujar en el pentagrama la 3ª y la 5ª que corresponden. Se pulsa en el pentagrama para colocar cada nota; los botones de alteración se usan antes de colocarla.

Tipos de acorde

Etiqueta Nombre 3ª 5ª
PM Perfecta Mayor 3ª Mayor (4 st) 5ª Justa (7 st)
Pm Perfecta menor 3ª menor (3 st) 5ª Justa (7 st)
5dis 5ª Disminuida 3ª menor (3 st) 5ª Dis. (6 st)
5Aum 5ª Aumentada 3ª Mayor (4 st) 5ª Aum. (8 st)

Ejercicio

Identificación de tríadas

El objetivo es reconocer el tipo de acorde a partir de las tres notas escritas en el pentagrama.

IDENTIFICAR TRÍADAS

Para consolidar los cambios de acorde conviene practicar al ritmo del metrónomo, empezando lento (50-60 BPM) y subiendo gradualmente. El afinador online permite verificar que cada nota del acorde suena afinada.

Preguntas frecuentes

¿Cómo se construye una tríada mayor desde cero?
Se escribe la fundamental en el pentagrama y se sube una tercera mayor (4 semitonos): dos posiciones hacia arriba con el accidente necesario para que haya 4 semitonos. Desde esa nota, se sube otra tercera menor (3 semitonos) para obtener la quinta justa. Ejemplo: Do → Mi (3ªM) → Sol (3ªm) = Do Mayor.
¿Cómo se construye una tríada menor?
Se escribe la fundamental y se sube una tercera menor (3 semitonos) para obtener la tercera. Luego se sube una tercera mayor (4 semitonos) para obtener la quinta. La quinta siempre será justa (7 semitonos desde la fundamental). Ejemplo: La → Do (3ªm) → Mi (3ªM) = La menor.
¿Cómo se construye una tríada disminuida?
Se escribe la fundamental y se sube una tercera menor (3 semitonos). Luego se sube otra tercera menor (3 semitonos más): el resultado es la quinta disminuida (6 semitonos desde la fundamental, un tritono). Ejemplo: Si → Re (3ªm) → Fa (3ªm) = Si disminuido.
¿Cómo se construye una tríada aumentada?
Se escribe la fundamental y se sube una tercera mayor (4 semitonos). Luego se sube otra tercera mayor (4 semitonos más): el resultado es la quinta aumentada (8 semitonos desde la fundamental). Ejemplo: Do → Mi (3ªM) → Sol♯ (3ªM) = Do aumentado.
¿Cómo verificar si una tríada construida es correcta?
Se verifican los dos intervalos desde el bajo: 1) Entre el bajo y la nota del medio debe haber el número correcto de semitonos (4 para 3ªM, 3 para 3ªm). 2) Entre el bajo y la nota superior debe haber 7 semitonos (justa), 6 (disminuida) u 8 (aumentada). Si ambos intervalos son correctos, la tríada está bien construida.
