Construir Escalas Mayores

Se muestra el nombre de una escala mayor. Tu tarea es dibujar las 8 notas en el pentagrama desde cero: tónica, 2ª, 3ª, 4ª, 5ª, 6ª, 7ª y octava. Usa los botones de alteración (♭ ♮ ♯) para añadir sostenidos o bemoles donde sea necesario.

Cómo usar el ejercicio: Haz clic (o arrastra) en el pentagrama para colocar cada nota. Selecciona primero la alteración con los botones ♭ ♮ ♯ si la nota lo necesita. Puedes usar Deshacer para borrar la última nota o Limpiar para empezar de nuevo. Las notas se ordenan automáticamente de grave a agudo.

El patrón de la escala mayor

Todas las escalas mayores siguen el mismo patrón de intervalos: T – T – S – T – T – T – S (tono, tono, semitono, tono, tono, tono, semitono). Aplicar este patrón a cualquier nota inicial produce la escala mayor correspondiente.

Grado 1ª 2ª 3ª 4ª 5ª 6ª 7ª 8ª
Intervalo — T T S T T T S
Do Mayor Do Re Mi Fa Sol La Si Do
Sol Mayor Sol La Si Do Re Mi Fa♯ Sol
Fa Mayor Fa Sol La Si♭ Do Re Mi Fa

Trucos para recordar las escalas

El círculo de quintas es la herramienta más útil: cada escala con sostenidos tiene un ♯ más que la anterior (Do → Sol → Re → La → Mi → Si → Fa♯ → Do♯), y cada escala con bemoles tiene un ♭ más (Do → Fa → Si♭ → Mi♭ → La♭ → Re♭ → Sol♭ → Do♭). Si conoces la armadura, deducir las notas alteradas es inmediato.
